 employee-performance-predictor
Machine Learning project to predict employee performance using real HR data with Streamlit dashboard



 📌 Overview

This project predicts **employee performance (Low / Medium / High)** using Machine Learning and provides an interactive **HR analytics dashboard** built with Streamlit.

 🎯 Problem Statement

Organizations struggle to:

* Identify high-performing employees
* Detect underperformers early
* Make data-driven HR decisions

This project solves that using **data analytics + ML**.

 💼 Business Value

✔ Identify top performers
✔ Improve promotion decisions
✔ Plan training programs
✔ Support HR analytics

🧠 How It Works

```text
Employee Data → Preprocessing → Feature Engineering → ML Model → Prediction → Dashboard
```

 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

📊 Dataset

* Real HR dataset (IBM HR style)
* Features:

  * Age
  * Department
  * Salary
  * Experience
  * Training Hours
